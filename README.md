# PIC_Assembler_proto_board
A experimental board for the book "A First Course in the Microcontroller Programing with PICmicro"

### 概要

CQ出版社の「はじめてのPICアセンブラ入門」のための実験用ボード（本の中ではユニバーサル基板に組んでいる回路を、プリント基板にしたもの）

詳しくはブログをご覧ください :

https://vita-brevis.hatenablog.com/entry/2025/12/15/085609

秋月のBタイプ基板（95 x 72mm）サイズで、「B基板用アクリルパネル」に合うようにしました。

- DC入力は5V
- スイッチは秋月で販売している"2MS1-T1-B4-M7-Q-E-S"です
- ネジ・スペーサーはM3で、スペーサーは基板の下は高さ5mm、基板の上は高さ25mmを使いました

###### 本の回路図と違う点

- PICkit3用のコネクタの実装
    - ISPできるようにした
- 電源周り
    - スイッチなどを実装
- サーボモーター用電源、小型モーター用電源とPIC用電源を繋ぐジャンパを設置
   - サーボモーターくらいなら電源共用でも動きました

###### 未検証の点

- USB-Cコネクタ周り（給電用に設けたが、使わなかった）
- モータードライバー TA7291P 周り（小型モーターを手に入れる前に読んでしまったため、実装していない）

###### 設計ミス、工夫の足りなかった点

Githubでも書きましたが、少し設計ミスや、工夫の足りなかった点があります。
（いないと思いますけど）もし発注するのであれば、適宜直してください。

- UARTはRS-232出力となっているが、TTLレベルの出力も用意しておけばよかった（FTDIのUSB-UART変換で読めるから。わざわざUSB-RS232アダプタ持ち出さなくてもいい）
- ADC用のVRを半固定抵抗で実装したが、いちいちドライバーを用意して回すのは手間なので、可変抵抗にしておけばよかった
- LCDとC7が干渉する
   - C7を低背のものにすれば解消する
- PICkit3のピンヘッダとRS-232のコネクタも干渉するが、これはもう仕方ない
- SWの番号の違い（本だと回路図は1, 2だが、ソースコード上は0, 1である。正直私のせいではない…）
- PIC16F877Aをゼロプレッシャー実装する
   - 今の基板はISP書き込みを念頭に置いている。が、ライターを使うならゼロプレッシャーになると思う
